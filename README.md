# XGuard - AI Insider Threat Detection System

XGuard is an AI-powered cybersecurity tool that detects potential insider threats in organizations using behavioral analytics and machine learning.

## Features
- Insider threat detection
- Risk score prediction
- Explainable AI alerts
- Interactive dashboard
- Automated PDF incident reports

## Tech Stack
- Python
- Streamlit
- Scikit-Learn
- Plotly
- FPDF

## Run Project

pip install -r requirements.txt

streamlit run dashboard.py
